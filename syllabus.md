## R Training Syllabus

* Session 1 - Introduction to R + Data Manipulations in R (9/12)

    * R data structures
    * RStudio layout + workflow (scripts, projects)
    * Objects
    * Functions
    * Packages
    * Reading in data
    * `dplyr` and `tidyr`

* Session 2 - Graphics and Document Creation with R Markdown (October)

    * `ggplot2` and/or `ggvis`
    * R Markdown syntax
    * R Markdown output formats (documents, presentations, HTML)
    * Mass producing output with loops
    * Optional workshopping time

* Session 3 - Interactive Web Applications with `shiny` (November)

    * `shiny` inputs and outputs
    * Reactivity basics
    * Reactive objects
    * `shiny` user interface
    * `shiny` in R Markdown documents
    * Optional workshopping time

* Session 4 - Intermediate `shiny` Programming (December)

    * Controlling reactivity (`isolate`, `observe`, `eventReactive`, `observeEvent`)
    * Dynamic input widgets with `renderUI`
    * Conditionally showing/hiding/toggling UI elements with `shinyjs`
    * Optional workshopping time
